# 약수의 합

## 문제 설명
정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

<br>

### 제한 사항
n은 0 이상 3000이하인 정수입니다.

<br>

### 입출력 예
| n  | return |
|----|--------|
| 12 | 28     |
| 5  | 6      |

<br>

## 나의 풀이
처음에는 n의 크기가 3000까지이므로, 1~n까지 for문을 돌려 0으로 나누어떨어지는지 확인하는 방식을 채택하였다. 그러나, 약수의 성질을 생각해보았을 때, n의 반 값만 for문을 돌려도 약수 값이 나올 수 있었다.
