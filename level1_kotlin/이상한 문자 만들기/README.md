# 이상한 문자 만들기

## 문제 설명
문자열 s는 한 개 이상의 단어로 구성되어 있습니다. 각 단어는 하나 이상의 공백문자로 구분되어 있습니다. 각 단어의 짝수번째 알파벳은 대문자로, 홀수번째 알파벳은 소문자로 바꾼 문자열을 리턴하는 함수, solution을 완성하세요.

<br>

### 제한 사항
문자열 전체의 짝/홀수 인덱스가 아니라, 단어(공백을 기준)별로 짝/홀수 인덱스를 판단해야합니다.<br>
첫 번째 글자는 0번째 인덱스로 보아 짝수번째 알파벳으로 처리해야 합니다.

<br>

### 입출력 예
| s                 | return            |
|-------------------|-------------------|
| "try hello world" | "TrY HeLlO WoRlD" |

<br>

## 나의 풀이
처음 split으로 문제를 해결해나가려고 하였으나, 지문의 공백이 여러개가 있을 수 있다고 하여, 문자열 원본을 해칠 것 같았다.
<br>
결국, check라는 임시변수를 두어 공백문자를 만나면 check를 초기화시켜주며 짝수이면 toUpperCase(), 홀수이면 toLowerCase()로 변형시켜 문제를 해결했다.
